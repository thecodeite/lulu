Structure.

The general template structure is the consistent throughout the template. Below we will outline typical HTML, CSS and Javascript file structure.

Template structure:
<!-- nav -->
<nav id="primary">
    <div class="boxed">
        <?php wp_nav_menu( array( 'sort_column' => 'menu_order', 'container_class' => 'nav', 'theme_location' => 'primary_menu' ) ); ?>
    </div>

    <a class="open" href="#menu-primary"></a>
</nav>
<!-- /nav -->
CSS structure:
/**
 * Nav
 */
nav#primary
{
    background: #3F3F49;
    width: 200px;
    height: 100%;
    padding: 25px 50px 0 0;
    position: fixed;
    top: 0;
    left: 0;

{

{

    font-weight: 400;

nav#primary ul li a:hover
{
 	color: #F16863;
}
/* @end Nav */
JS structure:
// primary drop menu
$("#primary li").hover(function() {
      $(this).find('ul:first').stop(true, true).animate({
            height: ['toggle', 'swing'],
            opacity: 'toggle'
      }, 300, 'linear');
});

Install.

There are two ways to install your new Wordpress theme. You may install via the Wordpress Admin section, or via FTP.

Installation via Admin:

Log in to your Wordpress Admin section at http://your-website.com/wp-admin/

Once logged in you will need to navigate to Appearance > Themes > Install Themes from the left menu. From there click Upload from the top sub-menu.

Click on the Browse button and browse to where you saved your theme and select it. Now click Install Now. Once installed it will return and ask you if you would like to activate the new theme, click activate.

Installation via FTP:

Open up your favourite FTP client and enter your FTP credentials given to you by your Web Host.

Once you have connected to your server via FTP you need to navigate to /wp-content/themes/

Next extract / unzip your theme.zip. This will create a folder with the same name as the theme.zip file only without the .zip extension.

Now you need to upload the extracted folder into your /wp-content/themes/ directory with your FTP client.

Once uploaded, head over to your Wordpress Admin section in your web browser and from the left menu navigate to Appearance > Themes. You will now see your newly installed theme. Click activate.

Top tip: When uploading your theme ensure you have extracted the theme from the download first and you are uploading only the theme, this will prevent you from getting the error "The theme is missing the style.css stylesheet".

Setup.

Create your pages from the Wordpress menu Pages > Add New (no need to add content yet).

Once you've created all of your pages, head to Appearance > Menus and create your Wordpress menu system. From the screen create a new menu and then from the Theme Locations box select Primary Menu and assign the menu you just created.

Scroll down and under Pages add whatever pages you'd like to display, these are the pages you created earlier. You can then drag and drop to order them and click save when you're done!

Next, head to Settings > Reading and set the page you'd like to display as your homepage and the page you'd like your Blog to display on.

Now go to Settings > Permalinks, we like to use a Custom Structure which is: /%category%/%postname%/ however, using the postname setting in there works fine too.

That's the basics done!

We'll go into more details on setting the menu system and page titles for this particular theme in the next section.

Options.

This theme uses the native Wordpress Customize section for configuring various options.

Let take a look. Head to Appearance > Customize from the menu. You will now see a live version of the theme with an options panel down the left hand side. Let's run through those sections.

  • Site title & Tagline: The title will display if no logo is set. The tagline will display on pages where no title is set or cannot be set.
  • Navigation: What navigation to use for the site. This will be the one you created earlier.
  • Static Frontpage: If you went through the setup process previously this will already be set.
  • Logo: Simply upload an image you'd like for your logo.
  • Styling: Choose an accent color that will carry through various part of your site such as hovers, links etc...
  • Social: Enter usernames / URL's for your social profiles, this will display the icons in the footer.

Note: Leaving any option blank will disable.

Portfolio.

This theme creates a custom post type for your portfolio, this will create a 'Portfolio' button in the left menu.

When you've created a page that will display the portfolio items be sure to set the page template to 'Portfolio' in the page attributes menu.

Simply give each item a title, content and a featured image. If you like to have multiple images simply add them in the content section.

You can also add a filter that will be used to filter the portfolio items on the portfolio page.

Note: You can use the Portfolio module as your homepage, or any other page.

Contact Page.

To use our contact page, simply create a page, set its page template to 'Contact' from the page attributes menu and add your content.

The email address that the mail will be sent to is the admin email set in Settings > General.

Installing sample data.

This theme comes with an XML file of the sample data from the theme demo. To use this data as a starting point is easy. From your wp-admin head to Tools > Import from the left menu.

On the next screen select Wordpress, you will then be presented with a popup asking you to install a plugin, go ahead and install it. Now from the Tools > Import menu again you will be able to browse and select the sample.xml that came with the themeand import it.

Shortcodes & Classes.

All of our themes are built around our popular CSS framework Workless.

So, not only do you get all the goodies we build into our themes, you also inherit the classes from Workless!

For those of you comfortable with CSS you can checkout the Workless site for full info and available options.

Ok, let's get on with it!

Columns.

You have a choice of columns for arranging your content, you have the following options:

one full, one half, one third, two third, one quarter and one fifth.

Using columns is very easy, you simply wrap any content you like with any column you like, you can also either use the shortcode variant or directly with a CSS class, let's have a look at the usage:

// Shortcode usage
[one half]
    you add your content in here, then close the shortcode.
[/one half]

// Class usage
<div class="one_half">
    you add your content in here, then close the element.
</div>
Pricing.

The next shortcode is probably the hardest to use as it has a few parameters, the pricing shortcode. Let's run through it:

// Each pricing box is used like this
[pricing heading="Demo" price="FREE" link="#" name="Sign Up" items="
    <ul>
	    <li>HTML Website</li>
	    <li>1 Business Cards</li>
	    <li>Branding Package</li>
	    <li>1 mths free hosting</li>
    </ul>
"]

You can see how this relates to the output on the frontend by checking the demo. You'll notice that in the demo we have 4 different pricing options, how do we set them up? Easy, we have just wrapped each pricing shortcode in a column shortcode, let's take a look:

// Wrap a pricing box inside a column
[one quarter]
    [pricing heading="Demo" price="FREE" link="#" name="Sign Up" items="
        <ul>
    	    <li>HTML Website</li>
    	    <li>1 Business Cards</li>
    	    <li>Branding Package</li>
    	    <li>1 mths free hosting</li>
        </ul>
    "]
[/one quarter ]

Troubleshooting.

  1. Q: I can't install the theme?
    A: Did you unzip it correctly? The theme comes in the main folder with the Documentation.
  2. Q: I can't seem to adjust my menu order?
    A: Did you create the Wordpress 3 menu outlined in 'Setup'?
  3. Q: When I go to my blog the posts don't show images but I added them?
    A: Did you add a 'Featured Image' to display?
  4. Q: The contact form doesn't seem to send messages?
    A: Did you add your email address in the 'General Settings' of the options panel?
  5. Q: I'm getting the error "The theme is missing the style.css stylesheet"?
    A: See #1.
